# Simple Node.js HTTP Server

This project is a basic **Node.js HTTP server** built using core Node.js modules.  


## Features

- Uses only **Node.js core modules** (`http`, `os`)
- Clean and styled HTML responses
- Multiple routes handling
- Displays system information
- Shows real-time server time
- Proper **404 Not Found** error handling

---

## Technologies Used

- **Node.js**
- `http` module – to create the server
- `os` module – to fetch system information
- HTML & CSS – for frontend rendering
